# GLBW — Graphics Library Basic Wrapper

Welcome to the GLBW documentation.

GLBW is a lightweight wrapper around OpenGL and [GLFW](https://www.glfw.org/) designed to make
window creation and basic 2D rendering easier for begginers.

GLBW is 100% pure C and should work in all enviroments that support OpenGL 3.0;

## Características principales
- API limpia y minimalista
- Dependencias internas ocultas (GLFW, GLAD)
- Ventanas multiplataforma
- Manejo de errores integrado
- Diseño profesional estilo SDK
